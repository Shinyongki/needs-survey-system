## 3. 다음 세션 작업 범위

### 목적
상반기 통합설문(S01, 61문항)과 하반기 통합설문(S02, 65문항)의 문항 내용을 검토한다.

### 확정된 선행 처리 사항
다음 두 가지는 이미 확정되어 있으므로 검토 시작 전 먼저 반영한다.

| 항목 | 처리 |
|---|---|
| 상반기 통합설문 섹션 F1 홍보사업 3문항 | 삭제 → 58문항 |
| 하반기 통합설문 섹션 F1 홍보사업 4문항 | 삭제 → 61문항 |

### 검토 관점

1. **현황분석 문항(현○○)과 욕구조사 문항의 역할 구분** — 경계가 모호한 문항 있는지 확인
2. **섹션 간 중복** — 유사한 내용이 다른 섹션에 반복되는지
3. **선택지 구성 적절성** — 현장 실태를 반영하지 못하는 선택지 있는지
4. **문항 논리 일관성** — 상반기(욕구)·하반기(결산) 구조에 맞는지
5. **문항 표현** — 응답자 입장에서 이해하기 어려운 표현 있는지

### 참고 파일 (프로젝트 지식)
- `상반기통합설문.md` — S01 61문항 전체
- `하반기통합설문.md` — S02 65문항 전체
- `현장설문_수정본.md` — 이번 세션 수정 완료본 (outputs 폴더)

- 객관적·논리적·비판적 시각을 적극적으로 제시한다 (동의에 기울지 않음)

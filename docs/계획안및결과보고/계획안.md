## 1. 추진 배경 및 필요성

### 1-1. 배경

- 사업 운영 근거 확보 및 실효성 강화를 위해 **현황분석·욕구조사** 체계 신규 도입
- 기존 계획안 구조에 데이터 기반 추진방향 근거 반영

| 구분 | 기존 | 변경 |
|---|---|---|
| 계획안 구조 | 개요 → 추진방향 → 세부계획 → 향후계획 | **현황분석 → 욕구조사 →** 추진방향 → 세부계획 → 향후계획 |
| 추진방향 근거 | 담당자 판단 중심 | **데이터 + 종사자 욕구** 기반 |

### 1-2. 필요성

- **현황분석**: 객관적 데이터로 현재 상황 파악 → "왜 이 사업이 필요한가" 증명
- **욕구조사**: 종사자 수요 직접 파악 → "무엇을 어떻게 해야 하는가" 방향 근거
- **사업 운영 근거 확보**: 계획안·결과보고서에 근거 문장 명시 필요

> 근거 문장 형식: **"○○조사 결과 ○○%가 ○○을 요청하여, ○○을 추진한다."**

## 2. 현황분석 체계

### 2-1. 개념 및 원칙

- **정의**: 사업 기획 전 현재 상황을 객관적 데이터로 파악하는 과정
- **핵심 원칙**: 단순 수치 나열이 아닌, **수치 → 시사점** 형태로 분석
- **데이터 출처**: 노인맞춤돌봄시스템 데이터, 전년도 결과보고서, 모니터링 기록, 설문 현황체감 문항, 현장메모
- **시기**: 매년 1~2월 (계획안 수립 전)

### 2-2. 현황분석 항목

| 분석 영역 | 주요 항목 | 활용 사업 |
|---|---|---|
| 수행기관 현황 | 기관 수, 신규기관 수, 운영 연차 분포 | 모니터링, 실무협의회, 전체 |
| 종사자 현황 | 전담사회복지사·생활지원사 수, 신규 입사·이직 현황 | 교육, 종사자 지원 |
| 서비스 제공 현황 | 대상자 수, 서비스 유형별 현황 | 전문지원, 직접서비스 |
| 전년도 점검 결과 | 지적사항 유형·빈도, 반복 지적 기관 수 | 모니터링, 컨설팅 |
| 전년도 교육 결과 | 참여율, 주제별 만족도 | 교육 |
| 상담·연계 실적 | 컨설팅·노무 상담 건수·유형, 위기사례·퇴원환자 연계 건수 | 전문지원 |
| 종사자 안전·심리 | 폭언·폭행 피해 건수, 심리 지원 요청 건수 | 심리지원교육, 종사자 지원 |
| 현장메모 | 사업 진행 중 현장 변동사항·특이사항 | 전체 |

### 2-3. 분석 방법

1. **데이터 수집** — 시스템·보고서·기록 등에서 항목별 수치 추출
2. **수치 분석** — 전년 대비 증감, 유형별 비율, 지역별 편차
3. **시사점 도출** — 수치 → 사업 강화 방향 연결
4. **계획안 반영** — 추진방향의 '현황 근거' 문장으로 명시

> ※ 2026년(첫해): 기존 행정 데이터(시스템 수치, 전년도 보고서 등) 활용  
> ※ 2027년 이후: 설문 현황체감 문항 결과 + 현장메모 추가 투입

## 3. 욕구조사 체계

### 3-1. 개념 및 원칙

- **정의**: 종사자가 광역지원기관에 어떤 지원을 원하는지 직접 파악
- **조사 대상**: 전담사회복지사, 생활지원사 (이용자 아님)
- **핵심 원칙**: 응답 비율(%)을 추진방향 근거 문장에 명시
- **결과 활용**: 당해 연도 사업 조정 + 다음 연도 계획안 추진방향 근거

### 3-2. 욕구조사 3종 체계

| 단계 | 조사명 | 시기 | 대상 | 주요 목적 |
|---|---|---|---|---|
| 1단계 | 상반기 통합 설문 | 4월 | 전담사회복지사 전원 | 기초 욕구 + 현황 체감 → 당해 계획 근거 |
| 2단계 | 현장설문 (16종) + 현장메모 | 사업 종료 직후 | 전담사회복지사 (J04·J06은 생활지원사) | 만족도·욕구 수렴 → 결과보고서 |
| 3단계 | 하반기 통합 설문 | 11~12월 | 전담사회복지사 전원 | 환류 → **다음 연도 계획안 근거** |

> ※ 현장설문: 도입부 3 + 공통 5(만족도) + 특화 5(욕구) = 13문항 × 16종  
> ※ J02(특이사항 보고): 행정 절차로 현장설문 대상 제외

### 3-3. 설문 응답 대상

| 직책 | 응답 설문 | 종수 |
|---|---|---|
| 전담사회복지사 | 상반기·하반기 통합설문 + J01·J03·J05·J07~J17 | 통합 2종 + 현장 14종 |
| 생활지원사 | J04(역량강화교육)·J06(심리지원교육) | 현장 2종 |

### 3-4. 단계별 세부 운영

#### ① 상반기 통합 설문

- 시행: 2026년 4월
- 방법: 온라인 설문 링크 배포
- 대상: 전담사회복지사 전원 (59개 기관)
- 활용: 당해 연도 사업 조정 근거

#### ② 현장설문 + 현장메모

- 시행: 각 사업 종료 후 즉시 (2주 이내 권장)
- 주체: 해당 사업 광역지원기관 담당자
- 대상: 16개 사업
- 구성: 도입부 3 + 공통 5(만족도) + 특화 5(욕구) = 13문항
- 현장메모: 변동사항·특이 반응·기관별 특이사항 별도 기록
- 활용: 결과보고서 반영 + 다음 연도 계획안 근거

#### ③ 하반기 통합 설문

- 시행: 2026년 11~12월
- 대상: 전담사회복지사 전원
- 활용: **2027년 계획안 추진방향 근거로 직접 반영**

### 3-5. 조사 결과 활용 흐름

```
상반기 통합 설문 (욕구 + 현황 체감)
  ↓ 당해 사업 추진방향 수립·실행
현장설문 (만족도 + 욕구) + 현장메모
  ↓ 결과보고서 반영
하반기 통합 설문 (욕구 + 현황 체감)
  ↓ 다음 연도 계획안 근거 명시
  ↓ (매년 반복)
```

## 4. 분석·활용 체계

|---|---|---|
| 기술통계 | 문항별 빈도·비율·평균 | 근거 문장 핵심 재료 |
| 만족도지수 | 100점 환산·사업 간 순위 | 사업별 성과 비교 |
| 근거문장 생성 | "○○% 요청 → ○○ 추진" | 계획안·결과보고서 직접 수록 |

### 4-2. 심화 분석

| 분석 | 내용 | 적용 시점 |
|---|---|---|
| **지역·규모별 비교** | 시군구별·규모별 응답 패턴 차이 | 2026년~ (즉시) |
| **이행 확인** | 상반기 욕구 → 현장설문 만족도 연결 | 2026년~ (즉시) |
| **연도 간 변화 추세** | 동일 문항 전년도 대비 변화 비교 | 2027년~ (2개년 필요) |

### 4-3. 연도별 분석 확대 로드맵

```
2026년 (첫해)
  ├─ 기본 분석: 기술통계 + 만족도지수 + 근거문장
  ├─ 지역·규모별 비교: 시군구별·규모별 욕구 차이
  └─ 이행 확인: 상반기 욕구 → 현장설문 만족도 연결

2027년 (2년차)
  ├─ 위 전체 +
  ├─ 연도 간 변화 추세: 2026 vs 2027 비교
  └─ 이행 확인 누적: 2026 이행 결과 → 2027 계획 반영

2028년~ (3년차 이후)
  ├─ 위 전체 +
  └─ 다년도 추세: 3개년 이상 시계열 분석
```

### 4-4. 이행 확인 구조

```
상반기 통합 설문: "○○% 요청"
  ↓ 사업 실행 시 반영
현장설문: 해당 사업 만족도 측정
  ↓
이행 확인: "○○% 요청 반영 → 만족도 ○.○점"
  ↓
결과보고서 수록 + 다음 연도 계획안 근거
```

### 4-5. 지역·규모별 비교 구조

| 규모별 (대형·중형·소형) | 규모별 만족도·욕구 차이 | 규모별 차별화 지원 |
| 기관 개별 | 특정 기관 응답 패턴 | 맞춤 컨설팅·점검 |

| 이행 확인 | "전년도 ○○ 요청 반영 → 만족도 ○.○점 → ○○ 유지/보완" | 2027년~ |

### 5-2. 섹션별 연결 구조

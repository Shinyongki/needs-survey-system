## 1. 추진 개요

### 1-1. 목적

- 광역지원기관 사업의 **현황분석·욕구조사 체계 신규 도입**
- 계획안 및 결과보고서에 **데이터 기반 추진 근거** 확보

### 1-2. 배경

| 구분 | 기존 | 변경 |
|---|---|---|
| 계획안 구조 | 개요 → 추진방향 → 세부계획 → 향후계획 | **현황분석 → 욕구조사 →** 추진방향 → 세부계획 → 향후계획 |
| 추진방향 근거 | 담당자 판단 중심 | **데이터 + 종사자 욕구** 기반 |

### 1-3. 사업 개요

| 구분 | 내용 |
|---|---|
| 시행 기관 | (재)경상남도사회서비스원 (광역지원기관) |
| 대상 기관 | 경남 18개 시군구 수행기관 59개소 |
| 조사 대상 | 전담사회복지사 전원 / 생활지원사 (역량강화·심리지원교육 한정) |
| 운영 체계 | 상반기 통합설문(4월, 56문항) + 현장설문 15종(연중, 10~12문항) + 하반기 통합설문(11~12월, 59문항) |
| 시행 시기 | 2026년 4월~ |
| 결과 활용 | 계획안·결과보고서 추진방향의 데이터 기반 근거로 활용 |

---

## 2. 운영 체계

### 2-1. 현황분석

- 매년 1~2월, 계획안 수립 전 실시
- 전년도 결과보고서·모니터링 기록·모인우리 기관 관리 데이터 등을 토대로 수행기관 현황, 종사자 현황, 서비스 실적 등 분석
- 분석 결과를 계획안 추진방향의 **현황 근거**로 활용

> ※ 2026년(첫해): 모인우리 기관 관리 데이터 및 전년도 결과보고서 활용  
> ※ 2027년 이후: 통합설문 현황분석 결과 + 현장메모 추가 투입

### 2-2. 욕구조사

- 종사자가 원하는 지원 내용을 직접 파악하여 추진방향 근거로 활용
- 3단계 설문 체계로 운영

| 단계 | 조사명 | 시기 | 결과 활용 |
|---|---|---|---|
| 1단계 | 상반기 통합설문 (56문항) | 4월 | 당해 연도 계획 근거 |
| 2단계 | 현장설문 15종 | 사업 종료 직후 | 결과보고서 |
| 3단계 | 하반기 통합설문 (59문항) | 11~12월 | 다음 연도 계획안 근거 |

### 2-3. 분석 및 결과 활용

- 분석 단위: 전체·권역별·연령대별·경력별 (기관별 분석 제외)
- 권역 구분: 북부(거창·함양·합천·산청) / 서부(진주·사천·고성·남해·하동) / 중부(창원·함안·의령·창녕) / 동부(김해·양산·밀양·통영·거제)
- 분석 결과를 토대로 **추진 근거 문장 생성** → 계획안·결과보고서에 수록

---

## 3. 기대효과

- **사업 운영 근거 확보**: 데이터 기반 추진 근거 명시
- **사업 실효성 향상**: 현장 수요 기반 교육 주제·지원 방식 결정
- **환류 사이클 완성**: 욕구 요청 → 반영 → 만족도 확인 → 다음 연도 반영
- **종사자 체감도 향상**: 욕구 결과가 실제 사업에 반영됨을 확인 가능

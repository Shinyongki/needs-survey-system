```

- 하현B3-1_소진경험 (1=자주, 2=가끔, 3=거의없음, 4=전혀없음)
- 하현B1-1_역량부족영역(복수) → 항목별 언급 비율

집단 간 차이가 두드러진 항목은 별도로 해석해줘.
```

## 유형 9. 상반기↔하반기 비교 분석

> ※ 이 지시어는 **두 파일을 모두 업로드한 대화**에서 사용하세요.

```
상반기·하반기 통합설문 데이터를 비교 분석해줘.
파일:
- 상반기: 상반기통합설문_[연도].csv  ← 실제 파일명으로 교체
- 하반기: 하반기통합설문_[연도].csv  ← 실제 파일명으로 교체

비교 문항 쌍:
① 심리적 소진 경험: 현B3-1 vs 하현B3-1
② 폭언·폭행 경험: 현B3-2 vs 하현B3-2
③ 위기사례 경험: 현D2-2 vs 하현D2-2
④ 서비스 인지율: 현E3-1 vs 하현E3-1
⑤ 게이트웨이 어려움: 현E1-1 vs 하현E1-1
⑥ 나눔사업 참여: 현E2-1 vs 하현E2-1

각 항목별로 상반기→하반기 수치 변화를 표로 정리하고,
악화·개선·유지 여부를 판단해줘.
변화가 두드러진 항목은 ⚠️ 또는 ✅ 표시해줘.
```

```
상반기·하반기 교육 관련 문항을 비교해줘.
파일:
- 상반기: 상반기통합설문_[연도].csv  ← 실제 파일명으로 교체
- 하반기: 하반기통합설문_[연도].csv  ← 실제 파일명으로 교체

비교 항목:
① 교육 방식 선호: B1-2 vs HB1-2
② 교육 주제 수요: B1-1(복수) vs HB1-1(복수) — 항목별 빈도 비교
③ 역량 부족 영역: 현B1-1(복수) vs 하현B1-1(복수)

상반기 대비 하반기에 수요가 증가한 교육 주제와
감소한 주제를 각각 정리해줘.
```

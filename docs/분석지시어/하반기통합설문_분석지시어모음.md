```

현황분석 문항 목록:

## 유형 9. 상반기↔하반기 비교 분석

```

파일:
- 상반기: 상반기통합설문_임의데이터_118명.csv
- 하반기: 하반기통합설문_임의데이터_118명.csv

비교 문항 쌍:
① 업무량 과중 인식: 현공-3 vs 하현공-3
② 심리적 소진 경험: 현B3-1 vs 하현B3-1
③ 위기사례 경험: 현D2-2 vs 하현D2-2
④ 서비스 인지율: 현E3-1 vs 하현E3-1

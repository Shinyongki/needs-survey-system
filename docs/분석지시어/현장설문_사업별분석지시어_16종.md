# 현장설문 — 사업별 결과보고서 분석 지시어 모음 (16종)

> **목적:** 각 사업 담당자가 하나의 사업을 마친 후 개별 결과보고서를 작성할 때 사용  
> **척도 방향:** 1=매우불만족 ~ 5=매우만족 (높을수록 좋음)  
> **긍정 응답:** 4+5 합산 / **부정 응답:** 1+2 합산  
> **분석 모델 유형:** 기술통계 · 만족도지수(CSI) · 갭(Gap) · 균형프로파일 · IPA · 욕구순위 · 텍스트 · 근거문장  
> 작성일: 2026-03-27

---

## 분석 모델 안내

각 사업별 지시어는 아래 8가지 분석 모델 중 해당 사업의 특성에 맞는 것들로 구성됩니다.

| 모델 | 이름 | 설명 |
|---|---|---|
| M1 | 기술통계 분석 | 문항별 빈도·비율·평균 집계표 |
| M2 | 만족도 지수(CSI) | 긍정비율(4+5), 100점 환산, 항목별 순위 |
| M3 | 균형 프로파일 | 다차원 척도를 레이더 차트로 균형 시각화 |
| M4 | 갭(Gap) 분석 | 항목 간 점수 차이, 올해 결과 vs 내년 수요 차이 |
| M5 | IPA 분석 | 중요도(공통4_업무도움 대리) × 만족도 4분면 매핑 |
| M6 | 욕구 순위 분석 | 선택형 문항 항목별 빈도·비율, Top 3 우선순위 |
| M7 | 텍스트 분석 | 주관식 키워드 추출, 긍정·개선요구 분류 |
| M8 | 근거문장 생성 | 분석 결과 → 경영평가용 사업 추진 근거문장 |

## J01. 수행기관 모니터링·점검
> 응답 대상: 전담사회복지사 60명 | 패턴: **프로세스 단계 평가형**  
> 핵심 구조: 점검(A1) → 결과통보(A2) → 개선지원(A3) 3단계 순차 평가  
> 현재 수치: 종합만족도 평균 4.02 / 긍정(4+5) 78.3%

### M1. 기술통계 분석
```
J01_모니터링점검.csv 데이터로 기술통계 분석을 해줘.
공통 5문항과 특화 척도 3문항(A1·A2·A3) 각각의
응답 분포(빈도·비율), 평균, 최빈값을 표로 정리해줘.
긍정(4+5) 비율과 부정(1+2) 비율을 각 문항마다 산출해줘.
```

### M2. 만족도 지수(CSI) 분석
```
J01_모니터링점검.csv 데이터로 만족도 지수를 산출해줘.
① 각 문항의 긍정비율(4+5 합산)을 CSI로 표현 (0~100점 환산: 긍정비율×100)
② 공통 5문항 + 특화 척도 3문항 총 8개 항목의 CSI 순위표 작성
③ 전체 평균 CSI 산출
④ CSI 80점 미만 항목에 ⚠️ 표시
```

### M3. 균형 프로파일 분석
```
J01_모니터링점검.csv 데이터로 균형 프로파일을 시각화해줘.
아래 8개 항목 평균값을 레이더 차트(방사형)로 표현해줘.
축: 사전안내 / 체계적운영 / 담당자전문성 / 업무도움 / 종합만족도 /
    점검항목반영(A1) / 결과통보명확(A2) / 개선지원충분(A3)
각 축에 평균값 레이블 표시.
공통 5문항(파란색)과 특화 3문항(주황색)을 색상으로 구분해줘.
```

### M4. 갭(Gap) 분석 — 프로세스 단계 병목 진단
```
J01_모니터링점검.csv 데이터로 점검 프로세스 3단계 갭 분석을 해줘.
3단계: A1_점검항목반영 → A2_결과통보명확 → A3_개선지원충분

① 각 단계 평균점수와 긍정비율을 순서대로 나열해줘
② 단계 간 점수 차이(갭) 계산:
   - 1→2단계 갭: A1 - A2
   - 2→3단계 갭: A2 - A3
③ 가장 큰 갭이 발생한 구간을 "프로세스 병목 지점"으로 표시
④ A3(개선지원충분)가 3단계 중 가장 낮다면, A4(내년강화항목)와 연결해서
   "병목 지점 → 내년 수요"로 이어지는 해석을 작성해줘
```

### M5. IPA 분석 — 중요도×만족도 4분면
```
J01_모니터링점검.csv 데이터로 IPA 분석을 해줘.
중요도 대리지표: 공통4_업무도움 (이 사업이 업무에 도움이 된 정도)
만족도 지표: A1_점검항목반영, A2_결과통보명확, A3_개선지원충분

① 각 항목의 (중요도 대리값, 만족도 값) 쌍을 산출
   - 중요도: 해당 항목 없이 공통4 전체 평균을 공유 기준으로 사용
   - 만족도: 각 특화 척도 평균
② 2×2 매트릭스로 구분:
   - 1사분면 (고중요·고만족): 유지 강점
   - 2사분면 (고중요·저만족): 집중 개선 필요 ⚠️
   - 3사분면 (저중요·저만족): 모니터링
   - 4사분면 (저중요·고만족): 과잉 투자 검토
③ 각 특화 항목이 어느 분면에 위치하는지 분류해줘
④ "집중 개선 필요" 항목이 있다면 내년 계획 반영 방향 제안
```

### M6. 욕구 순위 분석
```
J01_모니터링점검.csv의 A4_내년강화항목 문항을 분석해줘.
① 항목별 빈도·비율 표 (1위~5위 순서로)
② 1순위 항목 명시 + 비율
③ A3_개선지원충분 점수와 A4 1순위 항목이 논리적으로 연결되는지 해석
   (예: A3가 가장 낮은데 내년 강화 1위도 개선지원 관련이면 일관된 수요)
```

### M7. 텍스트 분석
```
J01_모니터링점검.csv의 A5_바라는점 주관식 응답을 분석해줘.
① 전체 응답에서 반복 키워드 Top 10 추출
② 키워드를 4가지 유형으로 분류:
   - 긍정 표현 (감사·도움·잘됨 등)
   - 점검 내용 관련 개선 요구
   - 사후 지원(개선지원·컨설팅) 관련 요구
   - 운영 방식(방문·서면 등) 관련 의견
③ 가장 많이 언급된 개선 요구 3가지를 문장으로 요약해줘
```

### M8. 근거문장 생성
```
J01_모니터링점검.csv 분석 결과를 바탕으로 결과보고서용 근거문장을 작성해줘.
아래 3가지 형식으로 각 1개씩 작성해줘.

형식 A (만족도 기반): "J01 조사 결과 종합만족도 ○○%가 긍정 응답하여, 현행 모니터링·점검 방식을 지속 운영한다."
형식 B (약점 개선): "점검 후 개선지원 충분성(A3) 평균 ○○점으로 3단계 중 가장 낮아, ○○을 강화한다."
형식 C (욕구 반영): "응답자의 ○○%가 ○○ 항목 강화를 요청하여, 내년도 ○○을 중점 추진한다."
실제 수치를 반드시 포함하고 문항명을 괄호로 병기해줘.
```

## J03. 역량강화교육 — 전담사회복지사
> 응답 대상: 전담사회복지사 60명 | 패턴: **교육 품질 3각 평가형**  
> 핵심 구조: 주제관련성(B1_1) · 강사전문성(B1_2) · 운영방식편리(B1_3) 3요소  
> 현재 수치: 종합만족도 평균 3.82 / 긍정(4+5) 65.0% ← 16개 사업 중 하위권

### M1. 기술통계 분석
```
J03_역량강화교육_전담.csv 데이터로 기술통계 분석을 해줘.
공통 5문항과 특화 척도 3문항(B1_1·B1_2·B1_3) 각각의
응답 분포(빈도·비율), 평균, 긍정비율(4+5)을 표로 정리해줘.
부정(1+2) 비율이 15% 이상인 문항에 ⚠️ 표시해줘.
```

### M2. 교육 품질 3요소 균형 분석
```
J03_역량강화교육_전담.csv 데이터로 교육 품질 3요소를 분석해줘.
3요소: B1_1_주제관련성 / B1_2_강사전문성 / B1_3_운영방식편리

① 3요소 평균점수와 긍정비율 비교표
② 3요소 중 가장 낮은 항목(교육 품질 약점) 특정
③ 3요소 간 최대 갭(가장 높은 요소 - 가장 낮은 요소) 산출
④ "교육 내용은 좋지만 강사가 문제인가, 방식이 문제인가, 주제가 문제인가"
   — 3요소 프로파일에서 개선 우선순위 도출
⑤ 레이더 차트: 3요소 + 종합만족도 + 업무도움 5개 축
```

### M3. 갭(Gap) 분석 — 공통 vs 특화 비교
```
J03_역량강화교육_전담.csv 데이터로 갭 분석을 해줘.

① 공통 5문항 평균 vs 특화 3문항 평균 비교
   (공통=운영 과정 평가 / 특화=교육 내용 품질 평가)
② 갭이 크다면 → 운영은 잘됐지만 내용이 부족한가, 또는 반대인가
③ 공통3_담당자전문성과 B1_2_강사전문성 비교
   (담당자 전문성 ≠ 강사 전문성일 경우 해석 필요)
④ 공통4_업무도움과 B1_1_주제관련성 비교
   (주제가 관련 있어도 실제 업무 도움으로 이어지지 않는 경우 확인)
```

### M4. IPA 분석 — 3요소 개선 우선순위
```
J03_역량강화교육_전담.csv 데이터로 IPA 분석을 해줘.
3요소(B1_1·B1_2·B1_3)에 대해:
- 중요도: B1_4_필요주제 복수선택에서 해당 영역 언급 빈도를 중요도 대리지표로 활용
  (예: '퇴원환자 서비스 실무' 많이 선택 → 관련 주제 중요도 높음)
- 만족도: 각 척도 평균

① 3요소를 중요도×만족도 매트릭스에 배치
② "집중 개선" 영역에 해당하는 요소 확인
③ 내년 교육 기획 시 어떤 요소에 자원을 집중해야 하는지 제안
```

### M5. 욕구 우선순위 분석 — 복수선택 심층
```
J03_역량강화교육_전담.csv의 B1_4_필요주제 복수선택을 분석해줘.
저장 방식: 세미콜론(;)으로 구분된 텍스트

① 세미콜론 파싱 후 항목별 언급 빈도·비율 집계 (빈도 내림차순)
② Top 3 주제 확정
③ 단독 선택 vs 복수 선택 중 자주 함께 선택된 주제 쌍 분석
   (예: "퇴원환자+특화위기"가 함께 선택되는 경우가 많다면 → 통합 커리큘럼 수요)
④ Top 3 주제와 이번 교육에서 다룬 주제가 다른 경우 → 미반영 수요 표시
⑤ 수평 막대차트로 시각화 (빈도순 정렬)
```

### M6. 텍스트 분석
```
J03_역량강화교육_전담.csv의 B1_5_바라는점 주관식을 분석해줘.
① 반복 키워드 Top 10 추출
② 교육 내용 관련 / 강사 관련 / 운영방식 관련 / 내년 요청 관련으로 분류
③ B1_2_강사전문성 점수와 강사 관련 불만 키워드 연결 해석
④ 내년 교육 기획에 직접 반영 가능한 구체적 요구사항 3가지 추출
```

### M7. 근거문장 생성
```
J03_역량강화교육_전담.csv 분석 결과로 근거문장을 작성해줘.

형식 A (지속): "역량강화교육 만족도 ○○%(긍정 응답)로 나타나, 전담사회복지사 대상 교육을 지속 운영한다."
형식 B (약점 개선): "3요소 중 ○○(평균○○점)이 가장 낮아, 내년도 ○○을 중점 개선한다."
형식 C (욕구 반영): "응답자의 ○○%가 ○○ 주제를 요청하여, 내년 교육 커리큘럼에 ○○을 포함한다."
```

## J04. 역량강화교육 — 생활지원사
> 응답 대상: 생활지원사 60명 | 패턴: **교육 품질 3각 평가형**  
> 핵심 구조: 주제관련성(B2_1) · 강사전문성(B2_2) · 운영방식편리(B2_3) + 단일선택 주제 수요  
> 현재 수치: 종합만족도 평균 4.00 / 긍정(4+5) 71.7%  
> J03과의 차이: 욕구문항이 복수선택 → **단일선택** (주제 1개만 선택)

### M1. 기술통계 분석
```
J04_역량강화교육_생활지원사.csv 데이터로 기술통계 분석을 해줘.
공통 5문항과 특화 척도 3문항(B2_1·B2_2·B2_3) 각각의
응답 분포(빈도·비율), 평균, 긍정비율(4+5)을 표로 정리해줘.
```

### M2. 교육 품질 3요소 균형 분석
```
J04_역량강화교육_생활지원사.csv 데이터로 교육 품질 3요소를 분석해줘.
3요소: B2_1_주제관련성 / B2_2_강사전문성 / B2_3_운영방식편리

① 3요소 평균·긍정비율 비교표
② 3요소 중 최고·최저 항목과 갭 산출
③ 생활지원사 특성 고려 해석:

   사업 자체는 도움이 되지만 종사자 처우 개선 효과는 별개
```

## 기타 응답 처리 안내

> ※ 구글폼에서 "기타 (직접 입력)"을 선택한 응답은 숫자 코드가 아닌 **자유 텍스트**로 CSV에 저장됩니다.

| 문항 유형 | CSV 저장 방식 | 처리 방법 |
|---|---|---|
| 단일선택 | 기타 선택 시 입력 텍스트 그대로 저장 | 숫자 코드 외 값 = 기타로 분류 |
| 복수선택 | `1;3;직접입력내용` (숫자+텍스트 혼재) | 세미콜론 파싱 후 숫자 항목·텍스트 항목 분리 |

**기타 처리 기본 원칙:**
- 기타 응답이 **5% 이상**이면 내용 요약 필수
- 기타 응답이 **3건 이상**이면 반복 키워드 추출
- 기타 내용에 새 카테고리가 보이면 별도 표시 — 다음 연도 선택지 보완 근거로 활용

## 유형 1. 현황분석 집계

```
상반기 통합설문 데이터로 현황분석 23개 문항을 집계해줘.
각 문항별로 응답 분포(빈도·비율)를 표로 정리하고,
핵심 수치(가장 높은 응답, 누적 비율 등)를 1~2줄로 해석해줘.
경고가 필요한 문항(부정 응답이 높은 항목)은 ⚠️ 표시해줘.

기타 응답 처리:
기타가 포함된 문항(현공-2_어려운영역, 현B전담-1_미참여이유, 현B1-1_역량부족영역)은
숫자 코드 외 텍스트 응답을 별도 집계하고,
기타 비율과 주요 내용을 표에 함께 표시해줘.
기타 비율이 5% 이상이면 ⚠️ 표시하고 내용을 1~2줄로 요약해줘.

현황분석 문항 목록:
현공-1_전년도만족도, 현공-2_어려운영역, 현공-3_업무량적절성,
현A-1_지적사항개선, 현A-2_이직잦음,
현C-1_협의회도움,
현D1-1_컨설팅이용,
현D2-1_특화대상자수, 현D2-2_위기사례경험,
현D3-1_협력체계원활,
현D4-1_노무어려움,
현B전담-1_미참여이유, 현B1-1_역량부족영역(복수),
현B3-1_소진경험, 현B3-2_폭언경험,
현E1-1_게이트웨이어려움, 현E1-2_돌봄로봇어려움,

## 유형 2. 섹션별 욕구 우선순위

```
상반기 통합설문 데이터로 섹션별 욕구 우선순위를 분석해줘.
단일선택 문항은 1순위 응답과 비율을,
복수선택 문항(세미콜론 구분)은 파싱해서 항목별 빈도 Top 3를 뽑아줘.
섹션 순서: A → C → D1 → D2 → D3 → D4 → B1 → B3 → E1 → E2 → E3 → F1 → F2 → F3
결과는 섹션별로 구분해서 표로 정리해줘.

기타 응답 처리:
- 단일선택 문항: 숫자 코드가 아닌 텍스트 응답은 '기타'로 집계하고 주요 내용을 요약해줘.
- 복수선택 문항: 세미콜론(;) 파싱 시 숫자가 아닌 항목을 '기타' 텍스트로 분류하고
  3건 이상이면 반복 키워드를 추출해줘.
- 기타가 1순위 또는 상위권이면 해당 문항의 선택지 보완 필요 신호로 해석해줘.
```

## 유형 3. 교차분석

```

분석 문항:
- 현B3-1_소진경험 (1=자주, 2=가끔, 3=거의없음, 4=전혀없음)
- 현B1-1_역량부족영역(복수) → 항목별 언급 비율
- B1-2_교육방식선호 (단일선택)
- 현공-3_업무량적절성

경력 집단별로 응답 분포를 비교하고,
집단 간 차이가 두드러진 항목은 별도로 해석해줘.
```

```

권역 구성:
  북부: 거창·함양·합천·산청 / 서부: 진주·사천·고성·남해·하동
  중부: 창원·함안·의령·창녕 / 동부: 김해·양산·밀양·통영·거제

분석 문항:
- 현공-3_업무량적절성

- 현D2-2_위기사례경험
- 현E1-1_게이트웨이어려움

권역별로 응답 분포를 비교하고,
특정 권역에서 집중 지원이 필요한 항목을 짚어줘.
지원 방식 권역 차등화가 필요한 근거도 제시해줘.

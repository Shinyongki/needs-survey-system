## 기타 응답 처리 안내

> ※ 구글폼에서 "기타 (직접 입력)"을 선택한 응답은 숫자 코드가 아닌 **자유 텍스트**로 CSV에 저장됩니다.

| 문항 유형 | CSV 저장 방식 | 처리 방법 |
|---|---|---|
| 단일선택 | 기타 선택 시 입력 텍스트 그대로 저장 | 숫자 코드 외 값 = 기타로 분류 |
| 복수선택 | `1;3;직접입력내용` (숫자+텍스트 혼재) | 세미콜론 파싱 후 숫자 항목·텍스트 항목 분리 |

**기타 처리 기본 원칙:**
- 기타 응답이 **5% 이상**이면 내용 요약 필수
- 기타 응답이 **3건 이상**이면 반복 키워드 추출
- 기타 내용에 새 카테고리가 보이면 별도 표시 — 다음 연도 선택지 보완 근거로 활용

**현장설문 기타 해당 문항:** B2-3(J04 교육주제), B4-4(J06 프로그램), D1-1(J08 컨설팅분야), D4-1(J11 노무유형), E3-2(J14 홍보채널)

## J01. 수행기관 모니터링·점검

> 응답 대상: 전담사회복지사 | 패턴: **프로세스 단계 평가형**
> 핵심 구조: 항목반영(A-1) · 피드백구체(A-2) · 개선방향파악(A-3) · 방식적합(A-4)
> 📌 분석 패턴 예시: 종합만족도 평균 4.02 / 긍정(4+5) 78.3%

### M1. 기술통계 분석
```
J01_모니터링점검.csv 데이터로 기술통계 분석을 해줘.
공통 4문항과 특화 척도 4문항(A-1·A-2·A-3·A-4) 각각의
응답 분포(빈도·비율), 평균, 최빈값을 표로 정리해줘.
긍정(4+5) 비율과 부정(1+2) 비율을 각 문항마다 산출해줘.
```

### M2. 만족도 지수(CSI) 분석
```
J01_모니터링점검.csv 데이터로 만족도 지수를 산출해줘.
① 각 문항의 긍정비율(4+5 합산)을 CSI로 표현 (0~100점 환산)
② 공통 4문항 + 특화 척도 4문항 총 8개 항목의 CSI 순위표 작성
③ 전체 평균 CSI 산출
④ CSI 80점 미만 항목에 ⚠️ 표시
```

### M3. 균형 프로파일 분석
```
J01_모니터링점검.csv 데이터로 균형 프로파일을 시각화해줘.
아래 8개 항목 평균값을 레이더 차트(방사형)로 표현해줘.
축: 사전안내충분(공통-1) / 현장파악충분(공통-2) / 운영개선도움(공통-3) / 종합만족도(공통-4) /
    점검항목반영(A-1) / 피드백구체(A-2) / 개선방향파악(A-3) / 방식적합(A-4)
공통 4문항(파란색)과 특화 4문항(주황색)을 색상으로 구분해줘.
```

### M4. 갭(Gap) 분석 — 사전안내 vs 사후결과

```
J01_모니터링점검.csv 데이터로 갭 분석을 해줘.

   점검 내용 반영도는 높지만 개선 방향을 스스로 파악하기 어려운 경우 분석
③ 가장 큰 갭 구간을 "개선 집중 지점"으로 표시
④ 갭 발생 원인 추정 및 내년 점검 운영 개선 방향 제안
```

### M1. 기술통계 분석
```
J03_역량강화교육_전담.csv 데이터로 기술통계 분석을 해줘.
```

### M2. 교육 품질 연계 분석
```
J03_역량강화교육_전담.csv 데이터로 교육 품질 연계를 분석해줘.

① 공통-1(주제내용적합) + 공통-2(강사전문성) → 공통-3(업무도움) 연결:

   도움은 됐지만 적용은 못한다면 → 실습·사례 중심 전환 수요 신호
④ 교육 품질 향상을 위한 내년 방향 제안
```

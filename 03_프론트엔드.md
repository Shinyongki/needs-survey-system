## 작업 목표

`C:\Projects\설문시스템\client\` 에 React(Vite) 앱을 구성하고

```bash
cd C:\Projects\설문시스템

```

## 화면 1: AgencyManager.jsx (`/agencies`)

**레이아웃:**
```
[검색창]  [지자체 필터▼]  [규모 필터▼]

| 기관코드 | 기관명 | 지자체 | 시설유형 | 규모 | 전담사회복지사 | 대상자 | 수정 |
|---------|-------|-------|---------|-----|-------------|------|-----|
| A48310001 | 거제노인통합지원센터 | 거제시 | 재가노인복지시설 | 소형 | 6명 | 571명 | [수정] |
...
```

**기능:**
- `GET /api/agencies` 로 목록 조회
- 지자체·규모 필터 (프론트에서 처리)
- 기관명 검색 (프론트에서 처리)
- [수정] 버튼 클릭 시 인라인 편집 (sw_count, ls_count, client_count만)
- 수정 저장 시 `PUT /api/agencies/:code` 호출

## 화면 2: CsvUpload.jsx (`/upload`)

**레이아웃:**
```
설문 선택: [S01 상반기통합설문 ▼]   연도: [2026 ▼]

┌──────────────────────────────────────┐
│   CSV 파일을 여기에 끌어다 놓으세요   │
│          또는 [파일 선택]             │
└──────────────────────────────────────┘

[파싱 미리보기 — 파일 선택 후 표시]
총 응답: 47행
기관 매칭: 47개 ✅
문항 수: 55개

              [취소]  [DB에 저장]
```

**기능:**
- `GET /api/sessions?year=2026` 로 회차 목록 조회 후 드롭다운
- 파일 선택/드래그앤드롭으로 CSV 로드
- 파일 선택 즉시 파싱 미리보기 표시 (실제 저장 전)
  - 총 응답 행 수
  - agency_code 추출 성공/실패 건수
  - 컬럼(문항) 수
- [DB에 저장] 클릭 시 `POST /api/responses/upload` 호출
- 저장 결과 표시: `삽입 47건 / 오류 0건`

## 화면 3: ResponseStatus.jsx (`/status`)

**레이아웃:**
```
연도: [2026 ▼]                        [CSV 내보내기]

           S01  S02  J01  J03  J04  J05  ...  J17
거제시
  거제노인통합지원센터   ✅   -    ✅   -    ✅  ...
  거제사랑노인복지센터   ✅   -    -    ✅   -   ...
거창군
  거창노인통합지원센터   ⚠️   -    ✅   ✅   ✅  ...
...

범례: ✅ 완료  ⚠️ 부분  — 미응답
```

**기능:**
- `GET /api/dashboard/status?year=2026` 로 데이터 조회
- 지자체별 그룹핑하여 표시
- 셀 색상: 완료=초록, 부분=노랑, 미응답=회색
- [CSV 내보내기] 클릭 시 현재 매트릭스를 CSV로 다운로드
